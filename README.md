## 🛠️ 使用方法

### 1️⃣ 安装依赖
```bash
npm install
```
2️⃣ 写文章

在 source/_posts/ 目录下新建 Markdown 文件，例如：

```bash
hexo new post "Spring事务解析"
```

3️⃣ 本地预览

```bash
hexo s
```
浏览器访问 http://localhost:4000

4️⃣ 部署到 GitHub Pages
```bash
hexo clean && hexo g && hexo d
```

```bash
.
├── _config.yml        # Hexo 配置文件
├── package.json       # 项目依赖
├── scaffolds/         # 模板
├── source/            # 博客文章和页面
│   └── _posts/        # 文章存放目录
├── themes/            # 主题目录
│   └── reimu/         # 使用的主题
└── README.md          # 项目说明

```
